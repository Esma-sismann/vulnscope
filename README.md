# vulnscope
"A GUI-based web vulnerability scanner for detecting XSS vulnerabilities."

🛡️ VulnScope: Web Vulnerability Scanner
VulnScope is a Python-based security tool designed to identify common web vulnerabilities like Cross-Site Scripting (XSS) through an intuitive graphical user interface (GUI).

✨ Features
🔍 Automated Form Discovery: Scans target URLs to find all HTML input forms.

🧪 XSS Testing: Automatically injects payloads to test for unsanitized inputs.

🖥️ User-Friendly GUI: Built with Tkinter for easy operation without command-line knowledge.

📊 Real-time Feedback: Provides instant scan results in the output console.

🚀 Getting Started
Install Dependencies:

Bash
pip install requests beautifulsoup4
Run the Application:

Bash
python VulnScope.py
